__UserController__

Handles:
1. Showing user profile
2. Editing/updating user info
3. (Optional) Admin management of users (list, delete, etc.)

__CartController__

Handles:
1. Showing current user’s cart
2. Adding/removing products
3. Updating quantity
4. Clearing cart after checkout

__WishlistController__

Handles:
1. Showing saved products
2. Adding/removing wishlist items

__TransactionController__

Handles:
1. Checkout process (creating a transaction from the cart)
2. Showing user’s past transactions
3. Showing transaction details (with items)
4. Possibly updating payment status (if admin stuff later)

__ProductController__

Handles:
1. Showing all products
2. Showing details of one product
3. (Optional admin side) creating/updating/deleting products
